FONTLOG for the Concrete font

This file provides detailed information on the Concrete Font Software.
This information should be distributed along with the Concrete font and any derivative works.

Basic Font Information

Concrete is a Unicode typeface that supports most languages that use the Latin script and its variants, and could be expanded to support other scripts.
